# Kalite Kontrol Listesi
## HTML Validasyonu
- [ ] HTML kodları geçerli ve hatasızdır.
## CSS Uyumlulugu
- [ ] CSS kodları tüm tarayıcılarla uyumludur.
## JS Hatalari
- [ ] JavaScript kodlarında hata yoktur.
## Responsive
- [ ] Sayfa mobil, tablet ve masaüstü cihazlarda doğru bir şekilde görünür.
## SEO
- [ ] Meta etiketleri doğru bir şekilde ayarlanmıştır.
- [ ] Başlık hiyerarşisi doğru bir şekilde uygulanmıştır.
- [ ] Görsellerin alt metinleri doğru bir şekilde ayarlanmıştır.
## Güvenlik
- [ ] XSS (Cross-Site Scripting) saldırılarına karşı önlem alınmıştır.
- [ ] CSRF (Cross-Site Request Forgery) saldırılarına karşı önlem alınmıştır.
## Performans
- [ ] Görseller optimize edilmiştir.
- [ ] Kodlar minimize edilmiştir.
## Erişilebilirlik
- [ ] Sayfa WCAG 2.1 AA standartlarına uygun bir şekilde tasarlanmıştır.
